Significado de la Tupla

  • Q — Conjunto finito de estados
  • Σ — Alfabeto: símbolos que puede leer
  • δ — Función de transición δ: Q×Σ→Q
  • q₀ — Estado inicial (q₀ ∈ Q)
  • F — Estados de aceptación (F ⊆ Q)
L(M) = { w ∈ Σ* | δ*(q₀,w) ∈ F }
El lenguaje del autómata es el conjunto de todas las cadenas que acepta.

AFD vs AFN

AFD (Determinista): para cada (estado, símbolo) existe exactamente una transición. δ: Q×Σ → Q

AFN (No Determinista): puede haber 0 o más transiciones. δ: Q×Σ → 2^Q

⚡ Todo AFN puede convertirse en un AFD equivalente (construcción de subconjuntos).
